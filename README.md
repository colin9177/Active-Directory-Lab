# Active Directory Lab

## Overview
I created a Windows Active Directory lab in Azure to simulate a business network.

## Tools Used
- Azure
- Windows Server
- Active Directory

## Steps I Completed
1. Created a virtual machine
2. Installed Active Directory
3. Added a client computer to the domain

## What I Learned
This lab helped me understand how domain controllers manage authentication.
